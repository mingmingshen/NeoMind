# 仪表盘

> 通过拖拽组件、实时数据绑定和公开分享功能，构建实时监控仪表盘。

---

## 打开仪表盘

1. 在顶部导航栏（桌面端）或可滚动标签页（移动端）中点击 **仪表盘**
2. 如果已有仪表盘，默认仪表盘会自动打开
3. 通过左侧侧边栏切换仪表盘：
   - 点击任意仪表盘名称即可打开
   - 当前仪表盘会高亮显示
   - 每个条目显示组件数量

如果尚未创建任何仪表盘，页面会显示空状态引导你创建第一个。

![仪表盘概览](../../img/dashboard_light.png)

> 图中标注：
> ① 侧边栏仪表盘列表（通过箭头按钮折叠/展开）
> ② 顶部栏中的仪表盘名称
> ③ **编辑** 按钮（铅笔图标）进入布局编辑模式
> ④ **分享** 按钮管理公开链接
> ⑤ 实时更新数据的组件网格

---

## 创建仪表盘

1. 点击仪表盘图标打开侧边栏，或从左边缘滑动打开（移动端）
2. 点击列表底部的 **"+ 新建仪表盘"** 按钮
3. 输入名称后按回车或点击勾号确认
4. 新仪表盘立即打开，显示空白画布

随时可以通过点击侧边栏中仪表盘名称旁的铅笔图标来重命名。要删除仪表盘，点击垃圾桶图标（会弹出确认对话框；最后一个仪表盘无法被删除）。

---

## 添加组件

1. 点击顶部栏中的 **编辑** 按钮（铅笔图标）进入编辑模式
2. 点击顶部出现的 **"+ 添加组件"** 按钮
3. 组件库面板从左侧滑入，按类别展示所有可用组件：

| 类别 | 组件 |
|------|------|
| **指示器** | 数值卡片、LED 指示灯、迷你趋势图、进度条 |
| **图表** | 折线图、面积图、柱状图、饼图 |
| **控制** | 命令按钮 |
| **展示** | 图片展示、图片历史、网页展示、Markdown 展示 |
| **空间** | 地图展示、视频展示、自定义图层 |
| **业务** | 代理监控、AI 分析师 |

4. 使用搜索栏或类别标签页找到所需组件
5. 点击组件将其添加到仪表盘 —— 它会出现在网格的下一个可用位置，使用默认尺寸

---

## 配置组件

添加组件后（或在编辑模式下点击已有组件），配置面板从右侧滑入，包含以下区域：

### 标题

设置组件头部显示的名称。

### 数据源

将组件绑定到实时数据。数据源选择器提供三个下拉菜单：

1. **来源类型** —— 可选项包括：设备、遥测、指标、命令、设备信息、扩展、扩展指标、扩展命令、系统、数据转换、AI 指标、代理
2. **实体** —— 从下拉菜单中选择具体的设备或扩展（自动从已连接设备和已安装扩展中填充）
3. **属性/指标** —— 选择或输入指标名称（例如 `temperature`、`humidity`）

常用来源类型：

| 来源类型 | 适用场景 | 绑定示例 |
|---------|---------|---------|
| 设备 | 读取已连接设备的指标 | 设备：`sensor-01` > `temperature` |
| 遥测 | 设备指标的时间序列历史 | 遥测：`sensor-01` > `humidity` |
| 扩展 | 来自已安装扩展的数据 | 扩展：`weather` > `temp` |
| 系统 | 平台健康指标 | 系统：`memory_percent` |

对于设备数据源，指标下拉菜单会根据设备类型模板自动填充。如果没有定义模板，可以手动输入指标名称。

### 显示选项（通用组件）

- **单位** —— 数值后显示的后缀（例如"C"、"%"、"kPa"）
- **格式** —— 数值格式字符串（例如"0.00"表示保留两位小数）
- **最小值/最大值** —— 图表组件的坐标轴范围，或进度条的范围
- **尺寸** —— 小、中、大预设
- **颜色** —— 组件主题色的 CSS 颜色值
- **前缀** —— 数值前显示的文本（例如"$"）

### 动作（通用组件）

配置组件上的点击动作，例如跳转到设备详情页或触发命令。

## 排列布局

在编辑模式下，可以自由排列网格上的组件：

- **移动**：点击并拖拽任意组件到新位置
- **调整大小**：拖拽组件边角或边缘的控制点改变尺寸；网格吸附确保整齐对齐
- **配置**：点击组件打开配置面板
- **删除**：点击组件右上角的关闭按钮

布局变更会自动保存。仪表盘以短延迟（500 毫秒）同步到服务器，因此快速拖拽操作会被合并为一次保存。

编辑完成后，点击顶部栏中的 **完成** 按钮（勾号图标）退出编辑模式。

## 分享仪表盘

1. 打开要分享的仪表盘
2. 点击顶部栏中的 **分享** 按钮
3. 分享管理器对话框打开，显示已有的分享链接（如果有）
4. 点击 **"+ 创建链接"** 生成新的分享链接
5. 配置链接：
   - **权限**：切换为 **只读**（查看者只能看到实时数据，无法交互）或 **可交互**（查看者可以调整时间范围、悬停查看详情）
   - **过期时间**：可选择永不过期、1 小时、24 小时、3 天、7 天或 30 天
6. 点击 **创建** 生成链接
7. 从列表中复制分享 URL —— 任何获得此链接的人无需登录即可查看仪表盘

### 管理分享链接

分享管理器列出所有活跃的链接，显示其权限级别、过期日期和完整 URL。对每个链接你可以：

- 使用复制按钮 **复制** URL
- 使用垃圾桶按钮 **撤销** 访问权限（链接立即失效）

分享的仪表盘每 30 秒轮询一次实时数据。可交互模式允许组件交互（更改时间范围、悬停提示），只读模式则展示静态的实时更新视图。

## 实用建议

- 先用 **数值卡片** 组件展示关键指标，再添加 **折线图** 展示趋势
- 使用 **迷你趋势图** 在数值展示旁显示紧凑的走势
- 将相关组件分组排列（例如将所有温室传感器放在一个区域）
- **AI 分析师** 组件提供内嵌聊天，用于探索仪表盘数据
- **代理监控** 组件可以直接在仪表盘上追踪 AI 代理的执行状态
- 展示大屏和看板使用 **可交互** 模式分享，报表使用 **只读** 模式
