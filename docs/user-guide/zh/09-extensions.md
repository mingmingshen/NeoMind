# 扩展

> 安装和管理扩展，为 NeoMind 平台添加天气预报、AI 视觉、OCR、视频分析等能力。

---

## 打开扩展页面

1. 在左侧导航栏点击 **扩展**，打开扩展管理页面
2. 页面以卡片网格形式展示所有已安装的扩展
3. 每张卡片显示扩展图标、名称、ID、版本、状态（运行中/错误）、命令数和指标数

![已安装扩展](../../img/extensions-installed.png)

> **上图说明**：扩展页面。每张卡片（①）显示扩展名称、状态标签和能力摘要（命令数和指标数）。鼠标悬停卡片可显示上下文菜单（②），提供详情、重新加载和卸载操作。

页面头部提供两个按钮：

- **扩展市场** -- 浏览并安装来自 NeoMind-Extensions 仓库的扩展
- **上传** -- 上传本地 `.nep` 扩展包文件

---

## 从扩展市场安装

### 第 1 步：打开扩展市场

1. 点击页面头部的 **扩展市场** 按钮
2. 打开全屏扩展市场对话框，自动从仓库加载可用扩展列表

### 第 2 步：浏览或搜索

1. 使用 **搜索栏** 按名称、描述或 ID 筛选扩展
2. 点击 **分类按钮**（如「AI」「数据源」）按类型筛选
3. 每张扩展卡片显示：名称、版本、描述、分类和作者
4. 已安装的扩展会显示 **已安装** 标签，不可重复点击

### 第 3 步：查看详情并安装

1. 点击扩展卡片上的 **查看详情**（或直接点击卡片）
2. 详情视图展示以下内容：
   - **工具** -- 扩展提供给 AI 智能体的工具
   - **指标** -- 数据字段，含数据类型和单位
   - **命令** -- 可从界面执行的命令
   - **依赖要求** -- 网络访问、所需 API Key 等
3. 点击对话框底部的 **安装** 按钮
4. 等待下载和安装完成
5. 对话框自动关闭，扩展出现在扩展页面中

## 配置已安装的扩展

### 第 1 步：打开扩展详情

1. 在扩展页面，点击扩展卡片上的 **详情** 按钮（或使用上下文菜单）
2. 打开全屏对话框，左侧边栏包含五个区域

### 第 2 步：配置设置

1. 在侧边栏中点击 **配置**
2. 配置表单根据扩展的 Schema 自动生成字段
3. 填写必要设置（常见字段包括 API Key、轮询间隔、端点地址等）
4. 点击 **保存并重新加载** 应用配置并重启扩展

### 扩展详情的其他区域

| 区域 | 用途 |
|------|------|
| **概览** | 基本信息（ID、名称、版本、状态、描述）、健康状态、文件路径 |
| **配置** | Schema 驱动的设置表单，保存后自动重新加载 |
| **命令** | 执行扩展命令，含参数表单和结果展示 |
| **指标** | 查看指标定义，可展开历史数据表格（1 小时 / 24 小时 / 7 天 / 30 天） |
| **日志** | 实时日志查看器，支持级别筛选、刷新和清空 |

## 扩展状态

| 状态 | 含义 |
|------|------|
| **运行中**（Active） | 扩展正在运行，正常提供数据 |
| **错误** | 扩展启动失败，请查看日志了解详情 |
| **已卸载** | 已从系统中完全移除 |

排查处于错误状态的扩展：打开扩展详情，进入 **日志** 区域查看错误信息。使用头部的 **重新加载** 按钮尝试重启。

## 可用扩展

| 扩展 | 说明 | 关键指标 | 需要配置 |
|------|------|---------|---------|
| **天气预报** | 通过 Open-Meteo API 获取实时天气 | `temp`、`humidity`、`wind_speed`、`precipitation` | 位置（经纬度） |
| **图像分析** | YOLOv11 图片目标检测 | `objects`、`count`、`classes` | 无 |
| **YOLO 视频** | 实时视频流检测 | `objects`、`count`、`fps` | RTSP/RTMP/HLS URL |
| **YOLO 设备推理** | 设备摄像头画面检测 | `objects`、`count` | 设备绑定 |
| **人脸识别** | ArcFace 人脸识别 | `faces`、`matches`、`confidence` | 人脸库 |
| **OCR 设备推理** | PP-OCRv4 文字识别 | `text`、`confidence` | 设备绑定 |
| **流媒体播放器** | 视频播放仪表盘组件 | （仅组件） | 流地址 |

### 扩展类型

- **数据源**：提供指标数据（如天气预报提供温度、湿度）
- **AI/视觉**：添加机器学习推理能力（如 YOLO 视频进行目标检测）
- **集成**：连接外部服务和协议
- **组件**：提供自定义仪表盘组件（如流媒体播放器）

## 使用扩展数据

扩展数据通过 DataSourceId 格式在整个 NeoMind 中可用：

```
extension:{extension_id}:{metric_name}
```

### DataSourceId 示例

| 数据源 ID | 扩展 | 字段 | 说明 |
|-----------|------|------|------|
| `extension:weather:temp` | 天气预报 | 温度 | 当前温度 |
| `extension:weather:humidity` | 天气预报 | 湿度 | 当前湿度百分比 |
| `extension:weather:wind_speed` | 天气预报 | 风速 | 当前风速 |
| `extension:yolo:count` | YOLO 检测 | 计数 | 最近一帧中的目标数 |
| `extension:yolo-video:count` | YOLO 视频 | 计数 | 最新帧中的目标数 |
| `extension:ocr:text` | OCR 设备推理 | 文本 | 最新提取的文字 |

### 在仪表盘中使用

1. 编辑仪表盘，点击 **添加组件**
2. 在组件的数据源设置中，输入扩展 DataSourceId（如 `extension:weather:temp`）
3. 配置显示选项（单位、阈值、颜色）
4. 保存组件，即可展示实时扩展数据

### 在规则中使用

在规则 DSL 中使用标准 DataSourceId 引用扩展指标：

```
RULE weather_rain_alert
  WHEN EXTENSION weather.condition = "Rain"
  DO
    NOTIFY(channel="telegram", message="检测到下雨！")
  END
END
```

### 在转换中使用

创建转换规则，将扩展数据与设备数据一起处理：

1. 前往 **自动化 > 数据转换**，创建新的转换
2. 将输入设置为扩展字段（如 `extension:weather:temp`）
3. 编写转换表达式
4. 输出将成为新的虚拟数据源，可在仪表盘和规则中使用

## 扩展管理

### 重新加载

使用 **重新加载**（卡片上下文菜单或详情页面头部）重启扩展，不会丢失配置。

### 卸载

1. 打开卡片上下文菜单（鼠标悬停卡片，点击三点图标）
2. 选择 **卸载**
3. 在对话框中确认操作 -- 此操作将删除所有扩展文件，且不可撤销

### 上传本地扩展

1. 点击页面头部的 **上传** 按钮
2. 选择一个 `.nep` 扩展包文件
3. 扩展自动安装并出现在页面中

---

## 使用建议

- **按需安装** -- 每个运行中的扩展都会占用系统资源
- **排查错误先看日志** -- 扩展处于错误状态时，先查看日志再考虑重新安装
- **修改配置后保存并重新加载** -- 配置变更在扩展重启后才会生效
- **频繁崩溃的扩展** 会被系统自动停止，防止资源耗尽
- **组合扩展与设备数据** 实现更强大的自动化（如天气 + 土壤湿度 = 智能灌溉）
